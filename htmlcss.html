HTML en CSS

HyperText Markup Language is een opmaaktaal van documenten die voornamelijk bedoeld is voor op het internet(www). Het is de standaard opmaaktaal voor webpagina’s. HTML word meestal bekeken als webpagina. Een HTML document kan lokaal worden opgeslagen maar kan ook via een server op het www geplaatst worden. Css is een opmaaktaal voor jouw website, met HTML en Css kan jij bepalen uit welke structuur jouw website bestaat en hoe deze eruitziet.
