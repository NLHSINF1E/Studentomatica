HTML

HyperText Markup Language is een opmaaktaal van documenten die voornamelijk bedoeld is voor op het internet(www). Het is de standaard opmaaktaal voor webpagina’s. HTML word meestal bekeken als webpagina. Een HTML document kan lokaal worden opgeslagen maar kan ook via een server op het www geplaatst worden. Elke webbapina bestaat uit HTML, dus eigenlijk kan deze opmaaktaal als onmisbaar worden beschouwd.

CSS

Css is een opmaaktaal voor jouw website, met HTML en Css kan jij bepalen uit welke structuur jouw website bestaat en hoe deze eruitziet. Na het opmaken van een HTML-pagina kunnen we de vormgeving van de verschillende elementen aan passen aan ieders wensen. Hier wordt CSS voor gebruikt. Kijk naar een krant, deze heeft een paginaformaat, binnen dat formaat moeten alle artikelen, koppen boven de artikelen en de fotos op een eigen plek worden gezet.
