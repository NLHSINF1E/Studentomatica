PHP

Hypertext Preprocessor is a script language meant to add dynamic to different webservers. The name of this language already tells the use of the script language, wicht is processing infromation to hypertext. PHP is really popular en is used by a lot of websites, forums, blogs and wiki's. PHP is seen as a part or more likely an addition of HTML. HTML takes care of the formatting while PHP takes care of the processing of information. With little pieces of PHP-code you can make a lot of usefull applications. Example: PHP can check if you properly filled in your name, adress and/or phone number in a contact form. These are the things that make PHP a script language instead of a program language.
